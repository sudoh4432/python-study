
## 条件付き書式を扱う

基本的な条件付き書式設定の流れ

* ランダム値のデータを生成
* シートのセルにデータをセット
* 条件付き書式を設定するための rule オブジェクトを作成する
* rule オブジェクトを摘要するセルの範囲を指定する

実際の条件付き書式のサンプル

* カラースケール
* データバー
* アイコンセット
* 値や数式を使った条件付き書式

Python でのデバッグの方法を少し

* pdb/ipdb を使ってみよう
